# 快速开始 - 通过微信消息添加定时任务

## 5分钟快速配置

### 第1步：配置管理员

1. 找到程序目录下的 `config.ini` 文件
2. 找到 `task_admin_list` 这一行
3. 填入你的微信名称（必须与微信中显示的完全一致）

```ini
task_admin_list = 你的微信名称
```

**示例：**
```ini
task_admin_list = 张三
```

如果有多个管理员，用逗号分隔：
```ini
task_admin_list = 张三,李四,王五
```

### 第2步：重启程序

保存配置文件后，重启 RuoYuBot 程序。

### 第3步：测试功能

向机器人发送私聊消息：

```
每日任务 09:00 测试消息
```

如果配置成功，你会收到类似这样的回复：

```
✅ 已成功添加每日任务
⏰ 时间: 09:00
📝 内容: 测试消息
```

## 查看和管理任务

1. 打开 RuoYuBot 程序界面
2. 切换到"定时任务"标签页
3. 在这里可以看到所有任务，包括通过消息添加的任务
4. 可以编辑、删除或禁用任务

## 常见问题

### Q1: 发送命令后没有反应？

**检查清单：**
- [ ] 你的微信名称是否在 `config.ini` 的 `task_admin_list` 中？
- [ ] 名称是否与微信中显示的完全一致（包括大小写）？
- [ ] 是否已重启程序？
- [ ] 是否是私聊消息（不是群聊）？

### Q2: 提示格式错误？

**常见错误：**
- 时间格式错误：应该是 `09:00` 而不是 `9点`
- 日期格式错误：应该是 `2026-02-10` 而不是 `2月10日`
- 星期格式错误：应该是 `周一` 而不是 `星期1`

**正确格式：**
```
单次任务 明天 15:30 提醒内容
每日任务 09:00 提醒内容
每周任务 周一 09:00 提醒内容
工作日 09:00 提醒内容
```

### Q3: 可以为其他人添加任务吗？

目前通过消息添加的任务只能发送给自己。如需为其他人添加任务，请使用程序界面。

### Q4: 如何删除任务？

打开程序界面，在"定时任务"标签页中找到对应任务，点击删除按钮。

### Q5: 任务什么时候生效？

任务添加后立即生效，无需重启程序。
